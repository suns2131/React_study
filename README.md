 ### DOM(Document Object Model)
 - 문서 객체 모델 <br>
    - 문서 객체모델이란?
    > HTML, XML 문서에 접근하기 위한 일종의 인터페이스<br> 
    > 문서내의 모든 요소를 객체로 정의하고 요소에 대한 접근 방법을 제공함.
